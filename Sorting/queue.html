📚 Queue Definition

Queue is a linear data structure that follows the First-In-First-Out (FIFO) principle.

Main Operations:

  1. Enqueue: Adds an item at the tail position
  2. Dequeue: Removes an item from the head position

Array implementation provides O(1) operations but has fixed size limitation.

💻 Implementation

Array-based queue uses head and tail pointers with circular wrapping.

class ArrayQueue { private int[] queue; private int head, tail, size; public void enqueue(int data) { queue[tail] = data; tail = (tail + 1) % capacity; size++; } public int dequeue() { int data = queue[head]; head = (head + 1) % capacity; size--; return data; } }

⚡ Complexity

Time Complexity:

  • • Enqueue: O(1)
  • • Dequeue: O(1)
  • • Peek: O(1)

Space Complexity: O(n)
