📚 What is a Queue?
A Queue follows FIFO (First In, First Out) principle. Elements are added at the tail and removed from the head.

Operations:
• Enqueue: Add element at tail
• Dequeue: Remove element from head
⚡ Complexity
Time Complexity:
• Enqueue: O(1) - Constant time
• Dequeue: O(1) - Constant time

Space Complexity: O(n) - Linear space
🎯 How to Use
1. Enter numbers separated by commas or spaces
2. Click Enqueue to add elements
3. Click Dequeue to remove elements
4. Use random buttons for quick testing
5. Adjust speed slider for animation speed

Queue Complexity Information

Array Implementation

Time Complexity:

  • Enqueue: O(1) - Constant time (adding at tail)
  • Dequeue: O(1) - Constant time (removing from head)
  • Peek: O(1) - Constant time (accessing head element)

Space Complexity: O(n) - Linear space (where n is the capacity)

Key Points

  • Array-based queues use circular buffering to maintain O(1) operations
  • The head and tail pointers wrap around using modulo arithmetic
  • Fixed capacity means no dynamic resizing (unlike linked list implementation)
